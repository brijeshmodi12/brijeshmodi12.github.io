Multi Document Text Summarization - A Survey

Amol Tandel, Brijesh Modi, Priyasha Gupta, Shreya Wagle, Sujata Khedkar
Conference Papers Accepted in IEEE International Conference on Data Mining and Advanced Computing - SAPIENCE 2016

Abstract

In today’s world, the daily hustle-bustle does not permit a human being to devote time for manually summarizing various lengthy documents. Hence it is of utmost importance to devise an application that will facilitate automated text summarization. Not only will this application save time but also render higher scope of efficiency. This application will allow the user to automatically summarize relevant information from various sources.
